### 📚 Highlighted Projects  

- **Adversarial Federated Learning Testing Framework**  
  Developing a research framework that simulates an online attacker in federated learning environments. The goal is to let the attacker learn from training behavior over time, discover patterns, and expose weaknesses in different FL aggregation defenses. This work focuses on stress-testing distributed learning systems under adaptive adversarial behavior rather than assuming a fixed attack strategy.

- **PID Control for Federated Learning**  
  Implemented adaptive aggregation logic using PID-inspired control to improve convergence stability and robustness under changing client behavior.

- **HAI Disclosure Experiment**  
  Designed a web-based experimental platform to study how AI versus human authorship cues influence user trust, perceived authenticity, and decision-making.

- **Spotify Scheduler Capstone Project**  
  Developed a React web application integrated with the Spotify API to support collaborative playlist scheduling and backend data flows.
